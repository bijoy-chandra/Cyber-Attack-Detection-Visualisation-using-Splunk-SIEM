## Deploying Splunk Enterprise Server on Windows 11

To Install splunk enterprise you have to register first into the splunk official portal. After registering login with the credential.

<img width="1817" height="865" alt="register " src="https://github.com/user-attachments/assets/56711679-0dc8-45dd-9fcb-d942967b1537" />

After login you'll see download option for splunk enterprise for windows.  

<img width="935" height="431" alt="download" src="https://github.com/user-attachments/assets/3da7dca9-9a38-40f9-b876-cbdfd97cce39" />


### Begin the installation

1. To start the installer, double-click the **splunk.msi** file. The installer runs and displays the Splunk Enterprise Installer panel.

<img width="450" height="347" alt="image" src="https://github.com/user-attachments/assets/3214e986-31ad-4fc4-9f06-dfef75b557aa" />

2. To continue the installation, check the "Check this box to accept the License Agreement" checkbox. This activates the "Customize Installation" and "Next" buttons.
3. (Optional) If you want to view the license agreement, click View License Agreement.
4. The installer displays the Logon Information: specify a **username** and **password** panel.

<img width="451" height="349" alt="image" src="https://github.com/user-attachments/assets/97fddb89-c6f5-4d2c-a921-87f68b4721b4" />

5. The installer displays the installation summary panel.

<img width="450" height="350" alt="image" src="https://github.com/user-attachments/assets/fd5c199e-20f4-457e-a4d2-7556032f8349" />

6. The installer runs, installs the software, and displays the Installation Complete panel.

<img width="444" height="343" alt="image" src="https://github.com/user-attachments/assets/67430647-d861-4069-8d6e-9a69dc9b3fdc" />


After completing the installation process, login with your credential you created during the installation process. After login the initial interface will look like this:
